# @jameswomack/api

Fastify API server for the pro-gram monorepo.

## Usage

```bash
pnpm --filter @jameswomack/api dev    # Dev server with hot reload
pnpm --filter @jameswomack/api build  # Compile TypeScript
pnpm --filter @jameswomack/api test   # Run tests
```

## Stack

- Fastify 5
- TypeScript (strict)
- Zod validation
